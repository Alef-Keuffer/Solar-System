Detailed Description

Data structure for Operations

The elem will be

  1. ⟨TRANSFORM⟩ ⟨float⟩ ⟨float⟩ ⟨float⟩ [float] or
  2. ⟨LOAD_MODEL⟩ ⟨NUMBER_OF_CHARACTERS⟩ ⟨char⟩ ... ⟨char⟩

and groups will be

  • ⟨BEGIN_GROUP⟩⟨BEGIN_GROUP⟩⃰ ⟨elem⟩⁺ ⟨END_GROUP⟩⃰ ⟨END_GROUP⟩

for example (inside a group),

LOAD_MODEL 7 'e' 'x' 'a' 'm' 'p' 'l' 'e'

our operations (world) will be

  • `⟨position⟩⟨lookAt⟩⟨up⟩⟨projection⟩⟨group⟩⁺

where position, lookAt, up and ⟨projection⟩ are

  • ⟨float⟩⟨float⟩⟨float⟩

we'll use an array of floats, therefore we will need to cast to char when reading the model filename characters and to int when reading the number of characters.
